# Plataforma E-Commerce 

Plataforma de e-commerce full stack desenvolvida do zero, cobrindo todo o fluxo de uma aplicação real: da interface do usuário à API e persistência de dados.

# Sobre o projeto
 
Este projeto foi construído para aplicar na prática conceitos de desenvolvimento full stack, com foco em arquitetura escalável, boas práticas de código e experiência do usuário. Inclui funcionalidades essenciais de um e-commerce moderno, como autenticação, catálogo de produtos e gerenciamento de dados.

# Tecnologias utilizadas

- **Front-end:** HTML5, CSS3, JavaScript
- **Back-end:** Node.js, Express
- **Autenticação:** JWT
- **Banco de dados:** MongoDB / Mongoose

## Funcionalidades

- Cadastro e autenticação de usuários
- Listagem e gerenciamento de produtos
- Carrinho de compras
- API RESTful para comunicação entre front-end e back-end
- Design responsivo

# Aprendizados

Durante o desenvolvimento, foquei em estruturar uma API robusta, lidar com autenticação segura via JWT e integrar o front-end de forma fluida com o back-end.
